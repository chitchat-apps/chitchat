ChitChat
============

A cross-platform [twitch.tv](https://twitch.tv) chat client.

Download
============
The lates version of the ChitChat client is available to download [here](https://chitchat-apps.github.io/#downloads).

Contributing
============
You can contribute to this repository by cloning and making a pull request.

Building
===========
Pre-requisites:

1. Rust installed.
1. Visit the Tauri website and follow the getting started guide [here](https://tauri.app/v1/guides/getting-started/prerequisites)
1. NPM / Node.JS installed
1. Run `yarn` / `npm install`
1. Build the project using `yarn tauri build` / `npm run tauri build`
1. Built files will be in the `src-tauri/target/` folder
